# vue-cli-test

> 基于vue-cli生成 
  axios 配置前端请求  
  express配置服务器  
  tedious连接sqlserver数据库 
  wepack 负责打包
  
  实现前后端的数据的交互Demo

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```
